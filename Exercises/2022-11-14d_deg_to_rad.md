## Instructions

Write a function that translates a value given in degrees to radians!

|--------------|----------------------------------------|
| name:        | `DegToRad()`                           |
| argument(s): | `x` `(numeric)`                        |
| return       | `numeric`, the value of `x` in radians |


*Hint: use the R builtin constant `pi` to get a good enough value of pi!*
