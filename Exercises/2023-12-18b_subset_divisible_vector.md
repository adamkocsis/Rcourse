## Instructions

We have this vector:
```R
v <- 100:1000
```
## Questions

Among the values presented here, 
- Exactly how many values are divisible by 16?
- Among the values that are divisble by 16, what is the 16th value?
- What is the index of this value in the original vector `v`?
